# Mario RL PPO — Local Training Suite

This repository contains:
- The Colab notebook (end-to-end training and evaluation)
- Generated report(s) in `reports/` (create via section G in the notebook)

## How to run (Colab)
1. Run A1 (pin NumPy) and restart the runtime.
2. Run A2–A3, then B-LOCAL.
3. Train via T (quick) or D (full).
4. Generate metrics (E/J/J2) and report (G).

Default action set: `SIMPLE_MOVEMENT`
Default reward mode: `spec` (+1 on forward step, -15 on death).
